# 代码书签

### 介绍

这是一个专门管理各种代码书签的项目，有一些是国外搜索引擎收集的，有一些是自己写的。

一般我们访问网站，采用的是 [http://](http:) 或 [https://](https:) 开头的协议，其实浏览器还支持其他协议。
比如 javascript: 伪协议，它的作用是：在当前页面执行一段自己准备的 javascript 代码。
这种操作类似于油猴脚本，不同的是，它不需要在每次页面加载后都执行，只需要在自己偶尔需要的时候点击执行就可以，而且因为免 UI 可以省去很多代码。

比如说，我想要获取 b 站当前观看视频的封面，直接点击事先准备好的代码书签即可，不必安装任何 Chrome 扩展，也不必安装脚本，更不必借助 http://www.galmoe.com/ 这样的专门网站获取封面。

### 使用

首先，下载 `bookmark.html` 文件

然后，在浏览器的**收藏夹/书签管理器** 导入 `bookmark.html` 文件即可



### 功能介绍

- 让图片飞：恶搞脚本，让当前网页所有图片飞出原来位置按一定轨迹摇摆

- reverse：恶搞脚本，让网页上下颠倒，向下滚网页上翻，向上滚网页下翻

- rewrite：让网页出现的所有文字变成可编辑状态，可以自己修改

- 记事本：让页面变成一个简易记事本

- 计算器：让页面变成一个简易计算器

- 小游戏：加载一只小飞机，通过方向键和空格键控制飞机消灭网页所有元素以通关(首次加载大概 15 s)

- 实时聊天：让同一时刻浏览同一网页的人彼此聊天

- b 站封面：跳转到当前播放视频的封面

- b 站弹幕：跳转到当前播放视频的弹幕

- 视频预览：跳转到当前播放视频的预览图

- b 站截图：截下当前视频画面的截图

- 截动态图：截下当前播放视频 3s 内的动态图

- 百度文库下载：解析百度文库当前文档的下载链接

- 视频解析：解析视频网站当前视频的真实地址并下载

- ibilibili：解析 bilibili 当前播放视频的下载地址

- 上传视频：用 b 站观看本地视频

- 实时弹幕：高亮当前视频所有 5s 内发送的弹幕

- 视频&弹幕上传：将本地视频和弹幕上传到 b 站观看

- 合成语音(英文)：将一句英文转换成音频，可下载，可播放

- 百度语音(中/英文)：将一句话转换成音频，可下载，不可播放
